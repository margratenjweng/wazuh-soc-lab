# Active Directory Administration Lab

## Project Overview

This project demonstrates the deployment and administration of an Active Directory environment using Windows Server 2019, Windows 11, VMware Workstation, and Wazuh SIEM.

## Technologies Used

- Windows Server 2019
- Windows 11
- Active Directory Domain Services (AD DS)
- Group Policy Management
- VMware Workstation
- Wazuh SIEM

## Project Activities

- Created Organizational Units (OUs)
- Created Security Groups
- Created Domain Users
- Joined Workstations to the Domain
- Configured Group Policy Objects (GPOs)
- Enabled Windows Security Auditing
- Generated and Investigated Event ID 4625
- Verified Event Collection in Wazuh

## Skills Demonstrated

- Active Directory Administration
- Group Policy Management
- Windows Security Auditing
- Event Log Analysis
- Wazuh SIEM Monitoring
- Security Event Investigation
